# Data folder

This project uses dummy data only. **Do not place real PHI or UCLA Health data in this repository.**

`data/dummy/sample_surveys.csv` contains example columns:
- `survey_id`
- `respondent_type` (patient / visitor / staff)
- `hospital`
- `department`
- `prompt` (which question)
- `response` (free-text)
- `rating` (numeric if present)
